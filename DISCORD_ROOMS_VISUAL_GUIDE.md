## ğŸ¯ Spacing Standards

### Container Padding
```
Large (p-6):  1.5rem (24px)  - Main containers
Medium (p-4): 1rem (16px)    - Card content
Small (p-3):  0.75rem (12px) - Items
Tiny (p-2):   0.5rem (8px)   - Dense content
```

### Gaps Between Items
```
Large (gap-4): 1rem (16px)    - Major sections
Medium (gap-3): 0.75rem (12px) - Group spacing
Small (gap-2): 0.5rem (8px)   - Item spacing
Tiny (gap-1):  0.25rem (4px)  - Tight spacing
```

## ğŸŠ Interactive Elements

### Hover Effects
```
Button Hover:
- Background: Darker shade
- Border: Lighter/colored
- Scale: Slight grow (1.02x)
- Transition: 150ms ease

Card Hover:
- Background: Darker with overlay
- Border: Brighter color
- Shadow: Enhanced
- Scale: Slight grow (1.02x)

Text Hover:
- Color: Change to accent
- Underline: Appear
- Transition: Smooth
```

### Active States
```
Pressed Button:
- Scale: 0.98x (compress)
- Opacity: 0.8
- Transition: 100ms

Active Toggle:
- Color: Bright green
- Icon: Filled/Changed
- Background: Dark shade

Disabled:
- Opacity: 0.5
- Cursor: Not-allowed
- Color: Grayed out
```
